按安全细分方向整理的面试题库，共十个方向，题目来源于历次真实面试的双向交流积累，持续更新。

面试时，理解题目背后的知识体系比背答案更重要。通过回答问题表达自己的逻辑思维和深入思考，才是让面试官印象深刻的方式。

## 按岗位方向选题

不同岗位的考察重点不同，以下是推荐的题目组合：

| 应聘岗位 | 重点方向 |
|---|---|
| 渗透测试工程师 | 渗透测试、应用安全 |
| 应用安全工程师 | 应用安全、渗透测试、安全研发 |
| 安全研发工程师 | 安全研发、应用安全、渗透测试 |
| 数据安全工程师 | 数据安全、安全合规 |
| 安全合规工程师 | 安全合规、数据安全 |
| 基础设施安全工程师 | 基础设施安全、渗透测试 |
| 安全蓝军/红队工程师 | 安全蓝军红队、渗透测试、应用安全 |
| 威胁感知与响应工程师 | 威胁感知与响应、基础设施安全 |
| AI 安全工程师 | AI 安全、应用安全、安全研发 |
| 安全管理/架构师 | 安全管理、所有方向各取一部分 |

各方向题库在此目录下，按需查阅。
